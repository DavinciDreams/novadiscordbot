# Nova Discord Bot

## Project Description
Nova is a Discord bot designed to enhance server management and user interaction through a variety of features. It integrates with OpenRouter API for advanced AI capabilities, providing users with smart responses and automation.

## Setup Instructions
1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file based on `.env.example`
4. Replace placeholder values with your actual tokens
5. Run the bot: `node main.js`

## Features
- Command handling system
- Database integration for persistent data storage
- AI-powered responses via OpenRouter API
- Utility commands for server management

## Usage Examples
**Basic Commands:**
```
!help - Display available commands
!ask [question] - Get AI-generated responses
```
